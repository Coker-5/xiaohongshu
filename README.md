# xiaohongshu

#### 介绍
小红书推荐数据采集程序，完成对了x-s参数的逆向。



#### 安装教程

1. 准备Python3.8,安装requests，execjs，pprint等第三方库。
2. 准备Node.js环境。
3. 运行main.js程序。
